# Instructions for Agents

This repository contains a Django project.

* Install all requirements for setting up the environment. Run `sh setup.sh`
* Use **ruff** for linting. Run `ruff check . --fix && ruff format .` before committing.
* The root `ruff.toml` defines styling rules such as 4-space indentation, double quotes and a line length of 120.
* Python version is 3.12
